# BController

Autonomous crypto spot trading for Home Assistant using Claude AI.

## Features

- BTC spot trading via Binance
- Claude Haiku for market analysis and trade decisions
- Risk management with stop-loss cascade
- Portfolio tracking via Home Assistant sensors
- Custom Lovelace dashboard card

## Installation

1. Add this repository as a custom repository in HACS
2. Select **Integration** as the category
3. Install **BController**
4. Restart Home Assistant
5. Add the integration via Settings > Devices & Services

## Requirements

- Binance account with API key (spot trading enabled)
- Anthropic API key
