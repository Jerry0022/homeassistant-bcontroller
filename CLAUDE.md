# BController — Home Assistant Custom Integration

Autonomous crypto spot trading integration for Home Assistant using Binance API and Claude API intelligence.

## Quick Reference

- **Stack:** Python 3.12+, Home Assistant custom component
- **Entry point:** `custom_components/bcontroller/`
- **Docs:** `docs/CONCEPT.md`, `docs/RULES.md`
- **Config flow:** HA UI-based (no YAML)
